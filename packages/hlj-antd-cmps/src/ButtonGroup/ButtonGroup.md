# ButtonGroup

> 使用方法

```
<ButtonGroup
    size="large"
    items={[
        {value:'0',text:'基本信息'},
        {value:'1',text:'活动商品'}
    ]}
    value={'0'}
    overCount={15}
    style={{marginBottom:10}}
    onChange={store.changeSource}
/>
```

1.  onChange 接受当前 value 作为参数
